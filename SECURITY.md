# Security Policy

RAVENLOCK es una herramienta defensiva de integridad local.

Puede:

- crear baseline local,
- crear archivos canario,
- escanear carpetas,
- generar reportes,
- detectar cambios masivos,
- usar modo TUI de vigilancia.

No debe incorporar:

- borrado automático,
- cifrado,
- persistencia oculta,
- telemetría obligatoria,
- envío de archivos a servicios externos,
- manipulación silenciosa del sistema.
